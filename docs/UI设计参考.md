## 🎨 设计规范建议

### 色彩系统

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色 | #409EFF | Element Blue，按钮、链接 |
| 成功 | #67C23A | 成功状态、空闲桌台 |
| 警告 | #E6A23C | 警告、待清台 |
| 危险 | #F56C6C | 错误、使用中桌台 |
| 背景 | #F5F7FA | 浅灰背景 |
| 深色 | #1A1A2E | 后厨页面背景 |

### 字体规范

| 层级 | 大小 | 用途 |
|------|------|------|
| 标题 | 24-32px | 页面标题、桌台号 |
| 重要数据 | 28-36px | 金额、数量 |
| 正文 | 14-16px | 普通文字 |
| 辅助 | 12px | 时间、备注 |

### 间距规范

- 卡片间距: 16-20px
- 内边距: 12-16px
- 按钮间距: 8-12px
- 列表项间距: 8px

### 优秀案例
1. **美团收银** - 简洁高效的收银界面

### 设计灵感
- 使用卡片式布局增加层次感
- 状态用颜色和图标双重标识
- 重要操作使用悬浮按钮
- 数据可视化使用简洁图表
